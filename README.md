# ClinMedKaz Payments — документация

Саб-сайт для приёма оплаты за публикацию статей журнала **ClinMedKaz**
([clinmedkaz.org](https://www.clinmedkaz.org/)). Основной сайт редактировать нельзя,
поэтому платёжный модуль вынесен в отдельное приложение и связывается с авторами
через персональные ссылки-приглашения.

Эти markdown-файлы в корне проекта — единый источник правды по продукту,
архитектуре и плану развития. Документы пронумерованы в порядке чтения.

## TL;DR для нового разработчика

```bash
cp .env.example .env          # заполнить ADMIN_*, HALYK_*, SMTP_* (или оставить SMTP пустым — письма в консоль)
npm install --prefix server
npm install --prefix client
npm run dev                   # Strapi на http://localhost:1337, React client на http://localhost:5173
```

- Публичная форма в dev: `http://localhost:5173/?invite=<id>` — открывается только по валидному приглашению.
- Админка в dev: `http://localhost:5173/admin` (логин `/admin/login`).
- Backend API: `http://localhost:1337/api/*`.
- Колбэки Halyk: `POST /api/halyk/postlink`.

> Все суммы и реквизиты — боевые юридические данные НИИ. Halyk-ключи нужно держать
> только в локальном `.env`; перед продакшеном замените их на боевые и **не коммитьте секреты**
> (см. [05-code-review.md](05-code-review.md), пункт P0).
